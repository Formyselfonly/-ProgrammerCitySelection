# -ProgrammerCitySelection
程序员城市选择的信息对比,包含每月收入,消费,生活指数等.涵盖绝大部分城市
## 数据真实性说明

- 数据来源:**Numbeo+livingcost**
- 数据:中位数结合平均数
- 我结合了我待过的各个城市的实际情况,发现这个数据非常真实,基本八九不离十.
- 我通过他们之间的对比确定了数据的真实性
- 通过其他网站的数据进行真实性佐证
- 数据源最后更新日期:March 01, 2024
- 希望有贡献者可以帮忙,帮助我及时更新,目前我提供的城市主要是程序员相关的发展城市
